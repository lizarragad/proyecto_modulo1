Introducción:
En este proyecto se presentara una empresa fabricante de cemento,
dicha empresa, contara con distintas plantas para producir dos clases dos
clases de cemento distintas, cada una de estas plantas tendran distintas
limitaciones en cuanto a la cantidad de unidades de cemento que podran producir,
además cada planta obtendra utilidades diferentes en cada una de las clases de
cemento que pueden producir.

Contando con estos recursos la empresa buscara, cual es la cantidad optima que
debera producir de cada clase de cemento de manera diaria, para asi lograr la 
utilidad máxima, respetando los limites diarios de demanda de cada clase de 
cemento a producir.
